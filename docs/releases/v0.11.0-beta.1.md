# Artifact v0.11.0-beta.1

Layer workflow and onboarding beta.

## Highlights

- Expanded blank-canvas starts for image, text, fill, noise, effect, recipes,
  examples, and random seeds.
- Added visible layer-row rename and quick-insert actions alongside duplicate,
  visibility, and delete.
- Added a second stack-only starter recipe and surfaced all layer recipes in
  the examples gallery.
- Preserved recovery drafts and confirmation around destructive random and
  recipe starts.
- Added a first-visit v0.11 guide entry that points users to the workflow
  guide.
- Clarified node-owned or unavailable controls in the layer inspector, including
  primitive camera ownership and full-frame noise placement.
- Kept graph areas as organizational folders in the layer workflow without
  changing render traversal.

## Validation

- `npm run format:check`
- `npm run lint`
- `npm run typecheck`
- `npm run test`
- `npm run build`
- `npm run test:browser`

## Known Beta Notes

- Browser/GPU rendering can vary slightly across machines.
- Browser tests cover focused starter, row-action, export, graph-area, and
  primitive-camera flows, not full visual regression.
- Project persistence remains local browser storage; account-backed persistence
  is still future work.
