# v0.35.0 Release Notes

Artifact v0.35.0 is the Masks, Transforms, Rotated Repeats, And Line Fields
release. It makes node workflows more practical for masked branches, movable
post-mask results, rotated motif systems, dirty alpha shadows, and procedural
warped line material.

This release changes the graph foundation for advanced visual recipes without
turning into a broad editor redesign, cloud release, project-history release, or
large effect batch.

Document schema, graph traversal, renderer/export behavior, and thumbnail
scheduling changed to support the new graph utility nodes and Line Field source.
AI scope, package export, and font policy remain unchanged.

## Highlights

- Added Mask Node with alpha, luma, and threshold modes so one branch can cut
  another before merge or export.
- Added Transform Node and Grime Shadow Node for moving completed branches and
  building layered dirty shadows from upstream alpha.
- Added Line Field source material plus Repeat per-copy rotation for warped
  posters, circular tokens, and deterministic repeated motifs.
- Updated Add Library, node search, recipes, docs, thumbnails, and export paths
  so the new workflow is available from the editor instead of only through
  fixtures.

## Scope Boundaries

- Included: serializable `maskNodes`, `transformNodes`, `grimeShadowNodes`, and
  `lineField` source layers with normalization for existing documents.
- Included: renderer, thumbnail, gallery, preset, and export parity for the new
  graph nodes and procedural source.
- Included: starter recipes for masked type lines, rotated circle tokens,
  warped line posters, and noise matte merges.
- Deferred: cloud sharing, project history, asset library, account features, and
  server-backed collaboration.
- Deferred: arbitrary vector line-field editing, freeform line displacement
  painting, and a broad new effect batch.

## Accepted Risks

- Line Field is a procedural source with built-in distortion modes, not a full
  editable vector field or hand-painted displacement system.
- Repeat per-copy rotation is deterministic by seed and node settings; it is not
  a per-copy manual timeline/editor.
- WebKit can emit benign ResizeObserver and Vite dynamic-import noise during
  local browser tests; the test harness filters those known non-product
  messages while still failing on product errors.
