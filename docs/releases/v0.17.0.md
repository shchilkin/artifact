# v0.17.0 Release Notes

Artifact v0.17 is the editor creative controls release. It keeps document
semantics, renderer/export behavior, graph traversal, thumbnail scheduling, and
AI alpha scope stable while making controls and add flows feel more like a
creative tool.

## Highlights

- Added formatted effect slider values with useful units such as `%`, `px`,
  `deg`, `bands`, and `steps`.
- Tuned Grain, Dither, Pixelate, and Misregistration defaults and ranges toward
  more usable creative starting points.
- Made Pixelate a clear low-resolution workflow through the existing focused
  effect system, without adding a new graph node kind or document field.
- Replaced the long Layers add dropdown with the shared Add Library used by
  Nodes.
- Added Add Library search, category groups, recent/popular starts, and
  hover/detail previews across Layers and Nodes.
- Added Node Add Library drag-to-canvas placement while preserving existing
  click-to-add behavior.
- Changed Add Library previews to use cached renderer output, so Fill, Text,
  Image, Noise, Split Tone, Pixelate, and other entries preview the same output
  model users get after adding them.
- Documented that Artifact should borrow mature SaaS mechanics such as search,
  keyboardability, predictable state, and accessibility without becoming a
  corporate/admin interface.
- Fixed the node-editor performance benchmark launcher so the benchmark can
  start React Router dev with explicit host and port arguments.

## Validation

Local validation passed on 2026-05-24:

- `npm run check`
- `npm run build`
- `npm run test:browser` across Chromium, Firefox, WebKit, mobile Chromium, and
  mobile WebKit. The full browser gate reported `174 passed, 1 skipped`.
- `npm run perf:node-editor`
- `git diff --check`

Additional coverage now includes:

- Add Library search, keyboard add, recent items, and rendered preview presence.
- Adding Pixelate from Layers and Nodes with shared formatted controls.
- Dragging an Add Library effect from the Nodes menu onto the canvas.
- Split Tone and other aliased presets resolving to valid rendered thumbnails.
- Effect control metadata, docs coverage, focused defaults, and random ranges.
- Graph-bootstrap layer insertion still connecting new Layers additions into
  the export path through the new Add Library flow.

## Accepted Risks

- Playwright WebKit is the automated Safari-family gate. A manual macOS Safari
  pass is still recommended before a broad public announcement.
- Known dev-server console noise may appear in WebKit during Vite route-module
  reloads and draft preview fallback startup, but it did not produce behavioral
  failures in the release gate.
- `npm run perf:node-editor` reported one dev-only `ERR_CONNECTION_REFUSED`
  console issue while the interaction metrics passed. Drag, slider, and pan
  scenarios had zero long tasks and p95 frame times around `17-18ms`.
- AI generation remains account/API dependent. This release does not expand AI
  product scope.
