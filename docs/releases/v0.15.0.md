# v0.15.0 Release Notes

Artifact v0.15 is the editor clarity and cross-browser confidence release. It
keeps document semantics, renderer behavior, graph traversal, thumbnails, and
export behavior stable while making the dark editor easier to read and safer to
ship across browsers.

## Highlights

- Added Playwright browser projects for Chromium, Firefox, WebKit, mobile
  Chromium, and mobile WebKit.
- Added focused browser scripts for running individual desktop browsers and
  mobile smoke coverage.
- Updated CI browser setup to install and run the expanded Playwright suite.
- Improved dark editor hierarchy for app background, workspace, panels, layer
  rows, canvas stage, node cards, graph areas, selected states, and toolbar
  actions.
- Moved the nodes Sign In action into the node toolbar with the other node
  actions.
- Gated AI generation diagnostics behind explicit debug flags while keeping one
  styled build/version console entry.
- Fixed layer text dragging so the preview does not switch into a skip-effects
  draft path during movement.
- Added a non-crashing primitive preview fallback for environments where WebGL2
  is unavailable.
- Preserved preview/export/render semantics for layer and node workflows.

## Validation

Local validation passed on 2026-05-23:

- `npm run check`
- `npm run build`
- `npm run test:browser` across Chromium, Firefox, WebKit, mobile Chromium, and
  mobile WebKit. The export download-event smoke runs in Chromium/WebKit; the
  Firefox CI job skips that single download event because GitHub Actions does
  not report it reliably.

Additional focused coverage now includes:

- Layer text dragging with effect stacks active during movement.
- Visual hierarchy assertions for panels, canvas, selected rows, node surfaces,
  selected nodes, graph areas, and toolbar actions.
- Mobile smoke for starter actions, layer list usability, action bar visibility,
  and horizontal overflow.
- Optional AI diagnostics remaining quiet by default and enabled only through
  explicit debug flags.

## Accepted Risks

- Playwright WebKit is the automated Safari-family gate. A manual macOS Safari
  pass is still recommended before a broad public announcement.
- Known dev-server console noise may appear in WebKit during Vite route-module
  reloads and draft preview fallback startup, but it did not produce behavioral
  failures in the release gate.
- `npm run perf:node-editor` was not rerun because this release did not change
  React Flow wiring, graph traversal, thumbnail scheduling, or node render
  semantics.
