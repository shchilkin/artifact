# v0.16.0 Release Notes

Artifact v0.16 is the editor workflow polish release. It keeps document
semantics, renderer behavior, graph traversal, thumbnails, project persistence,
AI alpha scope, and export behavior stable while making the editor state clearer
and the release/debug path quieter.

## Highlights

- Strengthened layer and folder states for selected, hovered, hidden, focused,
  draggable, disabled, active, and drop-target rows.
- Improved node editor contrast for selected nodes, muted nodes, handles,
  selected graph areas, toolbar actions, and canvas chrome.
- Added the first editor visual-system contract in
  `docs/editor-visual-system.md` and centralized foundation/semantic/node-card
  tokens in CSS.
- Added opt-in AI diagnostics behind explicit debug flags while keeping normal
  user sessions quiet.
- Kept a single styled build/version console entry with version and git SHA.
- Hardened browser assertions around asset-reference URL normalization and AI
  history flows.
- Opted GitHub JavaScript actions into the Node 24 runtime for the normal
  release gate.

## Validation

Local validation passed on 2026-05-23:

- `npm run check`
- `npm run build`
- `npm run test:browser` across Chromium, Firefox, WebKit, mobile Chromium, and
  mobile WebKit. The full browser gate reported `162 passed, 1 skipped`.

Additional coverage now includes:

- Layer selected/hidden/drop-target/folder visual states.
- Node selected state, graph area state, toolbar action state, and canvas
  contrast assertions.
- Optional AI diagnostics remaining hidden by default and visible only through
  explicit debug flags.
- AI generation history flow tolerant of stored asset references.

## Accepted Risks

- Playwright WebKit is the automated Safari-family gate. A manual macOS Safari
  pass is still recommended before a broad public announcement.
- Known dev-server console noise may appear in WebKit during Vite route-module
  reloads and draft preview fallback startup, but it did not produce behavioral
  failures in the release gate.
- `npm run perf:node-editor` was not rerun because this release did not change
  React Flow wiring, graph traversal, thumbnail scheduling, or node render
  semantics.
- AI generation remains account/API dependent. The diagnostics panel only shows
  safe presence/status fields and does not expose secrets.
