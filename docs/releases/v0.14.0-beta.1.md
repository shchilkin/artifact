# v0.14.0-beta.1 Release Notes

Artifact v0.14 is the editor/local-first beta line. It improves the layer
workflow, onboarding, and local editing confidence without depending on the
v0.13 private AI alpha infrastructure.

## Highlights

- Made layer mode faster for ordinary cover work with row-level rename,
  visibility, duplicate, delete, and quick-add actions.
- Improved graph-area presentation in the layer panel so areas read like
  lightweight folders while staying organization-only metadata.
- Added clearer empty states and blank-canvas starts for image, text, fill,
  noise, grain, examples, recipes, and random covers.
- Added layer-first recipe starts and exposed them through examples and docs.
- Added node-owned and unavailable-control notes for primitive camera framing,
  full-canvas noise placement, and graph-area organization.
- Added a first-cover guide entry from the blank-canvas start prompt.
- Completed Editor System Cleanup v1: layer panel feature folder, document
  command boundaries, generator orchestration extraction, and minor NodeCanvas
  incidental-state cleanup.
- Preserved renderer, graph traversal, thumbnail queue, export behavior, and
  preview/export semantics.

## Validation

Local validation passed on 2026-05-23:

- `npm run check`
- `npm run build`
- `npm run test:browser` with 49 browser tests passing

Additional focused coverage now includes:

- Layer row quick actions and row-level quick add.
- Empty layer panel quick starts.
- Layer-first recipe starts.
- Node-owned and unavailable layer-control notes.
- Graph-area folder summaries and dragging a layer out of an area.
- Node perf debug toggle persistence.

## Accepted Risks

- Package metadata and git tag still need to be cut as part of release
  packaging.
- Safari, Firefox, and mobile manual smoke are optional before public
  announcement, but recommended if the beta will be shared broadly.
- AI generation remains on the separate `v0.13.x-alpha` line and should not
  block this editor beta.
- `npm run perf:node-editor` was not rerun because this final release slice did
  not change React Flow wiring, thumbnail scheduling, graph traversal, or render
  scheduling.
