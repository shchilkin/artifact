# Artifact v0.1.0-beta.1

First public beta of Artifact, a browser-based album cover generator with layer
and node-based editing.

## Highlights

- Added release-ready CI checks for quality, build, and browser smoke tests.
- Improved node canvas layout, interaction stability, and primitive node
  behavior.
- Reworked renderer internals around shared stack and graph rendering paths.
- Added browser smoke tests for canvas switching, layer visibility, primitive
  controls, and export.
- Improved preview/export behavior for graph and layer workflows.
- Added production readiness, roadmap, architecture, and testing documentation.
- Added Biome, lint-staged, and Husky-based precommit code quality tooling.

## Beta Validation

- Deployed beta was used to create a real node-graph cover with image, text,
  effect, merge, and output nodes.
- The node editor produced a usable album-cover result end to end, which
  validates the core local-editor direction for the next beta cycle.

## Known Beta Notes

- Browser/GPU rendering can vary slightly across machines.
- Browser tests cover core smoke flows, not full visual regression yet.
- Project persistence is still browser/local-state oriented; server accounts
  and share links are roadmap items.
- Imported images are stored as document data URLs, which keeps documents
  portable but can make large projects heavy.
- Some advanced sticky-note ideas are documented but not yet implemented.

## Validation

- `npm run check`
- `npm run build`
- `npm run test:browser`
- GitHub Actions CI passed for the release tag, release PR, and merged
  `development` branch.
