# v0.27.0 Release Notes

Artifact v0.27 is a cumulative editor-confidence release. It includes the
already-merged v0.26 Layer Mode Polish work plus the v0.27 Editor Confidence and
Coverage slice.

The release keeps document schema, graph traversal, renderer/export behavior,
thumbnail scheduling, AI scope, package export, and font policy unchanged.

## Highlights

- Replaced the old Layers Add dropdown with the shared Add Library treatment:
  search, categories, hover previews, and click-to-add tuned for the linear
  Layers workflow.
- Improved empty Layers starts so a new document can move directly into useful
  sources, effects, recipes, projects, examples, or randomization.
- Improved layer row readability for selected, hidden, locked, and drag-over
  states in the dark editor.
- Fixed Layers reorder so layer order changes sync back to the canonical
  document and graph-backed layer order instead of becoming panel-only movement.
- Added shared editing target summaries in Layers and Nodes so selected source,
  effect, utility, output, hidden, locked, and output-path states are easier to
  read.
- Added soft lock guardrails using the existing layer-backed `locked` flag:
  locked layers resist deletion and reorder in Layers, and locked layer-backed
  nodes resist deletion in Nodes.
- Added repeatable Vitest coverage scripts for web, API, and combined coverage
  baselines.

## Accepted Risks

- `v0.26.0` was merged to `development` but not separately tagged. This
  `v0.27.0` tag is the public cumulative release for both v0.26 and v0.27.
- Coverage is a baseline and audit tool for now. Thresholds remain deferred
  until the signal is reviewed.
- Graph-only locking remains deferred; this release only uses the existing
  layer-backed lock flag.
- Visual regression snapshots for editor states remain future work.
- Server-backed share links, account-synced projects, and deeper font catalog
  sync remain future work.
