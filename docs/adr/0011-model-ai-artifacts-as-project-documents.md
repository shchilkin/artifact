# Model AI artifacts as project documents

Creative Directions and Compositions use the same `CanvasDocument` creative
payload as Layers and Nodes, but a Project-owned document record carries intent,
revision, and derivation metadata outside the renderer model. This preserves one
editable/renderable document contract while allowing a Project to keep a
direction, cover, poster, and later adaptations as separate linked artifacts;
converting one intent into another creates a derived document instead of
overwriting its source.
